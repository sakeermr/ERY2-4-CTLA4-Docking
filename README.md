## 🧬 Background

| Parameter | Value |
|-----------|-------|
| Target protein | CTLA-4 (immune checkpoint) |
| Natural binder | B7-1 (CD80) |
| Peptide | ERY2-4 (HLH scaffold, 40 aa) |
| Peptide KD | **196.8 ± 2.3 nM** (SPR, 25°C) |
| Inhibition IC₅₀ | **1.1 ± 0.03 μM** (SPR) |
| Cell-based IC₅₀ | **631.9 ± 167.9 nM** (DC assay) |
| CD28 selectivity | No cross-reactivity |
| Critical residue | **Trp33** (L33W mutation) |

ERY2-4 binds CTLA-4 at the **MYPPPY motif face** (residues 97–102), competitively 
displacing B7-1. The conserved L33W mutation across all affinity-matured variants 
identifies Trp33 as the primary hydrophobic anchor.

## 🔬 Pipeline Steps

```
PDB 1I8L → [Step 1] Extract CTLA-4 (chain C)
                         ↓
ERY2-4.fasta → [Step 2] Predict structure (PEP-FOLD3)
                         ↓ + Cys1–Cys40 disulfide
              [Step 3] Generate restraints
                         ├── AIR (MYPPPY motif + Trp33)
                         └── SS-bond (unambiguous)
                         ↓
              [Step 4] HADDOCK3 docking
                         ├── Rigid body (1000 models)
                         ├── Semi-flexible refinement (200)
                         └── Water refinement (50) → clusters
                         ↓
              [Step 5] Analysis
                         ├── Interface residues
                         ├── MYPPPY coverage
                         ├── Trp33 contacts
                         ├── B7-1 footprint overlap
                         └── PRODIGY ΔG prediction
                         ↓
              [Step 6] Visualization
                         ├── PyMOL sessions (.pse)
                         ├── Contact heatmap
                         └── Binding affinity plots
```

## ⚙️ Step-by-Step Usage

### Step 1: Prepare CTLA-4 Receptor
```bash
bash scripts/01_prepare_receptor.sh
# Output: data/output/01_receptor/ctla4_final.pdb
# Downloads 1I8L, extracts chain C, removes glycans/water
```

### Step 2: Prepare ERY2-4 Peptide
```bash
# Using PEP-FOLD3 (recommended)
python3 scripts/02_prepare_peptide.py --method pepfold3

# Using your own predicted structure
python3 scripts/02_prepare_peptide.py --input-pdb my_ery24.pdb

# Placeholder for testing
python3 scripts/02_prepare_peptide.py --method placeholder
```

### Step 3: Generate Restraints
```bash
python3 scripts/03_generate_restraints.py
# Generates:
#   - air_restraints.tbl (MYPPPY + Trp active residues)
#   - ssbond_restraints.tbl (Cys1–Cys40)
#   - trp33_restraints.tbl (Trp33 specific)
```

### Step 4: Run HADDOCK3
```bash
# Install HADDOCK3 first:
# git clone https://github.com/haddocking/haddock3.git
# cd haddock3 && pip install -e .

bash scripts/04_run_docking.sh
```

### Step 5: Analyze Results
```bash
python3 scripts/05_analyze_results.py
# Outputs analysis_report.txt with:
#   - Interface residues
#   - MYPPPY motif coverage
#   - Trp33 hydrophobic pocket insertion
#   - B7-1 footprint overlap %
#   - PRODIGY binding affinity prediction
```

### Step 6: Visualize
```bash
python3 scripts/06_visualize.py

# Open PyMOL sessions:
pymol data/output/06_visualization/01_overview.pml
pymol data/output/06_visualization/02_trp33_closeup.pml
pymol data/output/06_visualization/03_comparison.pml
```

## 🎯 Key Restraint Definitions

### CTLA-4 Active Residues (from 1I8L)
| Residue | Role | Part of MYPPPY? |
|---------|------|-----------------|
| Ile38 | Hydrophobic | No |
| Tyr48 | Hydrophobic (key pocket) | No |
| Glu50 | Electrostatic | No |
| Lys93 | Electrostatic | No |
| **Met97** | **Hydrophobic core** | **Yes** |
| **Tyr98** | **π-stacking** | **Yes** |
| **Pro99–101** | **MYPPPY** | **Yes** |
| **Tyr102** | **Key contact** | **Yes** |
| Asn110 | H-bond | No |

### ERY2-4 Active Residues
| Residue | Role |
|---------|------|
| Trp3 | N-helix anchor |
| Trp14 | Loop contact |
| **Trp33** | **PRIMARY anchor (L33W mutation)** |
| Trp39 | C-helix contact |

## 📊 Expected Results

Based on experimental data, successful docking should show:

- ✅ **MYPPPY coverage ≥ 50%** (ERY2-4 contacts residues 97–102)
- ✅ **Trp33 in hydrophobic pocket** (contacts Met97 and/or Tyr48)
- ✅ **B7-1 footprint overlap ≥ 50%** (explains competitive inhibition)
- ✅ **PRODIGY KD ~ 200 nM** (matching experimental 196.8 nM)
- ✅ **No CD28 contacts** (selectivity explained)

## 🧪 CI/CD

Every push triggers:
1. **Dependency check** — verify all tools available
2. **Receptor preparation** — download and process 1I8L
3. **Peptide preparation** — build ERY2-4 structure
4. **Restraint generation** — validate AIR and SS-bond files
5. **Analysis validation** — test interface analysis functions
6. **Visualization test** — verify PyMOL script generation

Full docking (Step 4) is triggered manually via GitHub Actions → 
*"Full Docking Pipeline"* workflow dispatch.
